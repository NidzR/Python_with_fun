---

### **Project 1: Talking Parrot**
**Kaam**: User ka input le kar usay "bolna" (text-to-speech) function perform karta hai.  
**Tareeqa**: `pyttsx3` library ko use kar ke input ko awaaz mein convert karta hai.  
**Required Installation**:  
- `pyttsx3` library:  
  ```bash
  pip install pyttsx3
  ```
**Purpose of Installation**: Text ko speech mein convert karne ke liye external library chahiye.

---

### **Project 2: Dice Rolling Simulator**
**Kaam**: Random number (1-6) generate karta hai, jaise dice roll karte waqt hota hai.  
**Tareeqa**: Python ka built-in `random.randint()` function use karta hai.  
**Required Installation**: None  
**Purpose**: Random number generation sikhata hai jo games ya probability-based projects mein kaam aata hai.

---

### **Project 3: Joke Generator**
**Kaam**: Random funny joke print karta hai.  
**Tareeqa**: `pyjokes` library ka `get_joke()` function use karta hai jo pre-defined jokes ka set provide karta hai.  
**Required Installation**:  
- `pyjokes` library:  
  ```bash
  pip install pyjokes
  ```
**Purpose of Installation**: Built-in jokes ko fetch karne ke liye library ki zarurat hoti hai.

---

### **Project 4: Guess the Number**
**Kaam**: Computer ek random number generate karta hai aur user ko usay guess karna hota hai.  
**Tareeqa**: `random.randint()` random number generate karta hai, aur conditionals (`if-else`) use kar ke user ka guess check kiya jata hai.  
**Required Installation**: None  
**Purpose**: Conditional logic aur user input handling sikhata hai.

---

### **Project 5: Emoji Translator**
**Kaam**: Text ko emojis mein translate karta hai.  
**Tareeqa**: Input text ko words mein split kar ke dictionary ke zariye matching emoji provide karta hai.  
**Required Installation**: None  
**Purpose**: Dictionary manipulation aur string processing ka practice karne ke liye.

---

### **Project 6: Magic 8-Ball**
**Kaam**: Random response (Yes/No/Maybe) return karta hai jo questions ka jawab dene ke liye use hota hai.  
**Tareeqa**: `random.choice()` function se pre-defined responses mein se ek response select karta hai.  
**Required Installation**: None  
**Purpose**: Random selection aur decision-making logic implement karna.

---

### **Project 7: Password Strength Checker**
**Kaam**: User ke password ki strength (weak/strong) check karta hai aur tips deta hai.  
**Tareeqa**: Python ka `re` (regular expressions) module use karta hai pattern matching ke liye.  
**Required Installation**: None (Built-in module `re`)  
**Purpose**: Password security aur regular expressions ka use sikhata hai.

---

### **Project 8: ASCII Art Generator**
**Kaam**: Text ko stylish ASCII art mein convert karta hai.  
**Tareeqa**: `art` library ke zariye pre-defined text art styles generate karta hai.  
**Required Installation**:  
- `art` library:  
  ```bash
  pip install art
  ```
**Purpose of Installation**: ASCII art ko create karne ke liye external styles aur fonts ka access chahiye.
